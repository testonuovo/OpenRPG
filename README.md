# 🎮 OpenRPG

OpenRPG is an open-source 2D RPG engine built in Python using Pygame.

The goal is to create a modular, easy-to-extend RPG framework that anyone can contribute to.

---

## 🚀 Current Version: v0.1

### Features:
- Player movement
- Tile-based map system
- Camera system
- Sprite rendering
- Basic collision system

---

## 🎯 Vision

OpenRPG aims to become a lightweight RPG engine where:

- Systems are modular
- Content is data-driven
- New mechanics are easy to add
- Community drives development
